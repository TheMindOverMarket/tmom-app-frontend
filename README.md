# TheMindOverMarket - Application Frontend

This repository houses the authenticated dashboard and trading interface for TheMindOverMarket.

## Architecture

- **Framework**: React + Vite
- **Language**: TypeScript
- **Charting**: TradingView Lightweight Charts
- **Styling**: Minimal CSS (Canonical)

## Application State

This is currently a foundation POC implementing the core candlestick chart renderer.

## Usage

```bash
npm install
npm run dev
```
